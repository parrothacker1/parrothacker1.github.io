Review

Parrot Security OS
It is the best hacking distro ever used.  It is smooth,lightweight and yeah it is more
It is designed to run on a 2gig RAM system so if u gotta 4 gig RAM perfect. But it do not give importance
to its looks.It has a simple UI. Also it has a Support Group where u can ask the errors you face and thus things made much more easy.  Only the thing is it do not support ARM devices thus it is not availabel to Pi users.  If you got a 4 gig RAM and wanna do pentesting then go for Parrot Security OS.  Its home version is also available.  Refer https://www.parrotsec.org for more details.



Arch Linux
It is a highly lightweight Distro. It can run on even a 1 gig RAM system
Only the hard part is that it's installation and setup is a bit hard and you need to be a master linux inorder to use it. But with the help of Google or DuckDuckGO you can set it up. And it has AUR repo from where you can install all tools.  If you customize it and install some of those hacking tools u want then , it will be a perfect distro for hacking.  Like for Parrot , it also has a Support Group where you can ask your errors and solve 'em.  As i said the only problem is it is not having a GUI installer(but can be installed from AUR.I haven't tried) and you gotta do all manually.
Visit https://www.archlinux.org for more details
Also there is a website called arch wiki where there is solutions for all erros and tutorials for programming,etc.



Kali Linux
Another hacking distro which is used all over the world. The OS which comes into mind when we hear "hacking". Now this distro is also good like Parrot Security OS.  But it needs about 8gig RAM to ensure smooth running. On a 4gig RAM system it get's headted up and when we start and IDE like VSCode or else,we feel like we are using Windows.  There is No support group for Kali Linux. Only way to solve errors is by Kali Forms.  And also Kali Linux is much more aware of their looks. They give importance to the sleek design and finish.
Kali Linux is officailly suported and mainted by Offensive Security. Kali has a wide variety of support. It supports ARM devices and also Mobile Phones.
Visit https://www.kali.org for more info


Kali Nethunter. The phone version of Kali Linux is now also becoming a trend. It has all the tools of Kali Linux and it is same like Kali Linux.  For Nethunter you need to root your phone and install the nethunter patched kernel for your phone.  Also you can use rootless Nethunter in Termux.  There is a Support Group for Nethunter where you can ask your doubts and errors and get them solved.
For more info visit https://www.kali.org/kali-linux-nethunter/ for more info.



Debian
Like Arch Linux,Debain is also lightweight and it is easy to install. Both the Parrot Security OS and Kali Linux is based on Debian OS. Debian is easy to use and if you are new to linux,I recommend you to start with Debain(You may see to start from Ubuntu on search engines but Debian is the best to start)  It can run on a 1gig RAM system and it is smooth. Like Arch Linux,Debian also have a wiki where you can solutions for your errors and also learn programming,etc.  Visit https://www.debian.org for more details.Also like ArchWiki there is also wiki for debian.



Tails OS
It is a debian based distro designed for high anomality. It erases all files before it switches off thus leaved behind no traces.  It starts Tor by default and it creates a new user at every boot even if persistent. It only save browser files,all other files are wiped. It is mainly used by journalists,criminals,etc like people.  You can access darkweb from that OS.Actually to access darkweb , there is no need of tails. Install tor and use.  Tails is used if you are somewhere else and you want to use a PC and you want to leave no evidences that you have used. You can also configure a liveusb and use Tails.  For more info visit https://tails.boum.org/ for more details regarding it
